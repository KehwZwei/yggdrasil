> __该模块定义整个`Yggdrasil`项目的依赖__

---

## i. 功能
- 约束外部依赖的版本号
- 统一`Yggdrasil`各个模块的构建过程
- 统一`Yggdrasil`各个模块的版本

---

## ii. 使用

__1. 通过`parent pom`引入__
```xml
<dependency>
    <groupId>org.s3s3l.yggdrasil</groupId>    <!-- 导入yggdrasil parent pom -->
    <artifactId>yggdrasil-parent</artifactId>
    <version>3.5.0-RELEASE</version>
    <type>pom</type>
    <scope>import</scope>
</dependency>
```

__2. 通过在`pom.xml`中指定parent来引入__
```xml
<parent>
    <groupId>org.s3s3l.yggdrasil</groupId>
    <artifactId>yggdrasil-parent</artifactId>
    <version>3.5.0-RELEASE</version>
</parent>
```
