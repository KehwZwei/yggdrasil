# __yggdrasil__
> A batch of tool-kits to help coding faster &amp; easier

## __Usage__

``` xml

<dependencyManagement>
    <dependencies>
        <dependency>
            <groupId>io.github.s3s3l</groupId>
            <artifactId>yggdrasil-parent</artifactId>
            <version>3.6.1-RELEASE</version>
            <type>pom</type>
            <scope>import</scope>
        </dependency>
    </dependencies>
</dependencyManagement>
```
